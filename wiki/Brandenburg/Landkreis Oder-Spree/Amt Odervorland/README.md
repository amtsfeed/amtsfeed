# Amt Odervorland

Website: https://www.amt-odervorland.de

## Kein Scraper vorhanden

Die Website des Amtes Odervorland läuft auf TYPO3, bietet aber keine strukturierten Listen für Nachrichten, Veranstaltungen oder Amtsblätter an. Inhalte werden als statische HTML-Blöcke direkt in Seiten eingebettet (TYPO3 Content Elements) ohne maschinell auswertbare Struktur (keine `<time>`-Elemente, keine einheitlichen CSS-Klassen, keine Datum-Felder).

Die Navigation enthält keine dedizierten Seiten für:
- Aktuelles / Pressemitteilungen
- Veranstaltungskalender
- Amtsblatt

Sobald die Website eine strukturierte Nachrichtenoder Veranstaltungsseite erhält, kann hier ein Scraper ergänzt werden.
