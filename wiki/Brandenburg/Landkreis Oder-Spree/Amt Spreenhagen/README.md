# Amt Spreenhagen

Amt im Landkreis Oder-Spree mit News, Veranstaltungen und Amtsblatt.
Quelle: https://www.amt-spreenhagen.de

## Quellen

| Typ       | URL |
|-----------|-----|
| News      | https://www.amt-spreenhagen.de/news/1 |
| Events    | https://www.amt-spreenhagen.de/veranstaltungen/index.php |
| Amtsblatt | https://www.amt-spreenhagen.de/amtsblatt/index.php |

## Datenqualität

- **News:** 18 Einträge
- **Events:** 130 Einträge
- **Amtsblatt:** 1 Einträge

## Besonderheiten

- CMS: **PortUNA (Verwaltungsportal)**
- Amtsblatt im alten PortUNA-Tabellenformat (`<td>Nr. N/YYYY</td>`)
- Events mit exakten Uhrzeiten aus `<time>` Tags (`event-entry`-Format)
